One-time data migration → Supabase

Standalone tool, not linked from the main app. Delete this file once migration is verified complete.

In the OLD app's browser devtools console, run copy(JSON.stringify(records)) and paste the result into the first box below. Do the same with copy(JSON.stringify(fttbRecords)) for the second box. Dates are converted from dd/mm/yyyy to ISO automatically. Any record with an embedded base64 pdf field is uploaded to Supabase Storage as part of the same pass.
