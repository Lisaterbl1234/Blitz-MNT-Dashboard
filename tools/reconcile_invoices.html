Reconcile Invoice Register → MNT Dashboard

Standalone tool, not linked from the main app. One-time reconciliation for the batch of 68 invoices from the June 2026 invoice register. Delete this file once the reconciliation is verified complete.

Matches each invoice row against live mnt_records using the customer job/ticket number (the digits before the FRG reference) plus an exact amount match. Only rows with exactly one confident match are auto-selected. Nothing is written until you review the preview and click "Apply Selected Updates" — for a matched row this sets status to Invoiced and appends the invoice/PO numbers to the notes field (existing notes are preserved, never overwritten); for an unmatched row (no record found on the dashboard at all) this creates a brand-new record already marked Invoiced with the PO/invoice numbers in its notes, since that invoice has no matching PO/quote row otherwise.
